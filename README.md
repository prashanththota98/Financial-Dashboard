Project Overview
This repository containsa Personal Finance Dashboaed built with React. The dashboard hepls users to track theirfinances, visual trends, and gain insights into thier spending habits

key Features

- Balance Trend Chart: Shows monthly income vs. expenses
- Category Pie Chart: Visualizes expense distribution by category
- Transaction List: Search, filter, sort, edit, delete, and export transactions
- Insights: Displays key metrics like highest income month, top spending area, and savings efficiency
- Dark/Light Mode: Toggle between themes for better usability

Tech Stack

- Frontend: React, JavaScript, TailwindCSS
- Charts: Recharts
- State Management: React Context API
- Utilities: LocalStorage for persistence, Intl.NumberFormat for currency formatting
